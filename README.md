# janus-plugin-rs

Library for creating Rust plugins to [Janus](https://janus.conf.meetecho.com/). Still highly unstable, so not published.

``` toml
[dependencies]
janus-plugin = { git = "https://github.com/mquander/janus-plugin-rs" }
```

## Building

```
$ cargo build
```

## Testing

```
$ cargo test
```
